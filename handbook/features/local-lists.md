# Múltiplas Listas Locais

Esta página resume a etapa que transformou o Aisly em um fluxo real de listas.

## O que foi entregue

- criação de listas
- renomeação
- arquivamento
- separação entre listas ativas e arquivadas
- persistência local das listas

## Por que isso importa

Essa etapa tornou o app útil para rotinas reais de compra e preparou a base para itens, orçamento e recorrência.

## Como validar

1. Rode os testes unitários.
2. Abra o app.
3. Crie pelo menos duas listas.
4. Renomeie uma delas.
5. Arquive outra.
6. Confirme a separação entre listas ativas e arquivadas.
